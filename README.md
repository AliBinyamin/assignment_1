# Personal Website - Ali Fahad Binyamin

## Project Overview
This is a professional personal website built with HTML5 and CSS3, featuring three main pages:
- **CV Page (index.html)**: Introduction, education, skills, work experience, and projects
- **Class Schedule (schedule.html)**: Current semester schedule and completed courses
- **Contact Page (contact.html)**: Contact form for visitors

## Features Implemented
✅ Responsive design with CSS Grid/Flexbox layout  
✅ Background images from Subtle Patterns  
✅ Sticky navigation header  
✅ Profile image with animated border  
✅ CSS pseudo-elements (::first-letter, ::before, ::after)  
✅ Fragment links with custom styling  
✅ Email and phone URI schemes (mailto:, tel:)  
✅ Social media icons with hover effects  
✅ Table with rowspan/colspan for schedule  
✅ Contact form with HTML5 validation  
✅ Hover animations and transitions  
✅ Mobile-responsive design  

## Setup Instructions

### 1. Webhook Setup for Contact Form
1. Visit https://webhook.site
2. Copy the unique URL provided
3. Open `contact.html`
4. Replace `your-unique-url-here` in the form action with your webhook URL

### 2. Profile Image
- Replace the placeholder image URL in all HTML files with your actual profile photo
- Current placeholder: `https://via.placeholder.com/150x150/3498db/ffffff?text=AFB`
- Recommended size: 150x150px for best results

### 3. Social Media Links
- Update the Twitter and LinkedIn URLs in all HTML files with your actual profiles
